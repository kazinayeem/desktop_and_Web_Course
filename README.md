## Overview
This assignment includes beginner-friendly PHP and JavaScript problems. Code is simple and written in small parts based on each requirement. No code comments are included inside source files.

## Files
- `php_basics.php`: PHP basics tasks 1-15.
- `php_oop.php`: PHP OOP tasks 1-5.
- `js_basics.js`: JavaScript problems 1-15 (console-based).
- `index.html`: Sample HTML for DOM problems 16-20.
- `your-script-file.js`: JavaScript for DOM problems 16-20.

## How to run
- PHP files: run with any PHP interpreter and view printed outputs in the terminal.
- JavaScript basics: open a browser console and load `js_basics.js` or run with Node.js.
- DOM tasks: open `index.html` in a browser.
